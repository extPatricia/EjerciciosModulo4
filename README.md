# EjerciciosModulo4

## Primer ejercicio con estructura POO
Tema: “Criaturas Musicales”
Un pequeño sistema donde las criaturas del mundo producen sonidos que se combinan en una sinfonía mágica.
1. MusicalCreature → una criatura que puede emitir sonidos.
2. Melody → una composición creada por varias criaturas.
3. Conductor → el director que organiza a las criaturas y las hace tocar juntas.
